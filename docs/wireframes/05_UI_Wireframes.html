Design Principles

Visual Elements:

  • Header bars: ============================================
  • Success messages: ✓ Success!
  • Error messages: ✗ Error!
  • Warning messages: ⚠ Warning!
  • Input prompts: _ (underscore)
  • Separators: --------------------------------------------

Navigation:

  • Number selection for menu options
  • 'B' or 'Back' to return to previous menu
  • 'Q' or 'Quit' to exit current view
  • 'Y/N' for confirmations
  • ENTER to continue after messages

Input Formats:

  • Animal ID: Auto-generated number (e.g., 1, 2, 3)
  • Keeper ID: Auto-generated number (e.g., 1, 2, 3)
  • Cage ID: Auto-generated number (e.g., 1, 2, 3)
  • Dates: DD-MM-YYYY
  • Phone: Standard UK format

Error Handling:

  • Clear error messages
  • Suggested actions
  • Input validation before processing
  • Confirmation prompts for critical operations
